Данный репозиторий содержит исходный код дипломного проекта — десктопного приложения для автоматизации работы сотрудника, принимающего и обрабатывающего заявки от граждан. Система разработана с прицелом на использование в профильных технологических и государственных учреждениях, включая ООО «Специальный Технологический Центр».

Проект написан на **C# (WPF)** под платформу **.NET 10.0** и использует реляционную базу данных **MS SQL Server** для надежного хранения информации. 

## ✨ Ключевой функционал
Система предоставляет полный цикл управления обращениями и включает в себя следующие модули:

### 🔐 Безопасность и авторизация
* Вход в систему по логину и паролю.
* Хранение паролей в базе данных в виде SHA-256 хешей для обеспечения безопасности.
* **Журнал аудита:** автоматическое логирование всех ключевых действий сотрудников (вход, создание, редактирование, удаление записей) с визуальной подсветкой типа операции.

### 📋 Управление обращениями (Заявками)
* Создание, чтение, обновление и удаление (CRUD) заявок.
* Просмотр подробной карточки обращения с историей изменения статусов.
* Продвинутая фильтрация заявок по ФИО заявителя, категории проблемы и временному периоду.

### 👥 Ведение базы заявителей (Граждан)
* Учет контактных данных (ФИО, телефон, Email).
* Разделение интерфейса и логики с применением архитектурного паттерна **MVVM** (`CitizensViewModel`, `RelayCommand`).

### 📊 Аналитика и отчетность
* Построение столбчатых диаграмм по количеству обращений в разрезе категорий.
* Экспорт статистических данных в табличные форматы (Excel/CSV).
* Генерация и печать индивидуальной карточки обращения через COM-интеграцию с **Microsoft Word** (`Microsoft.Office.Interop.Word`).

### 🗂 Справочники и Администрирование
* Динамическое управление категориями проблем и статусами через отдельное окно.
* Управление учетными записями персонала, назначение ролей и смена паролей доступа.

## 🛠 Стек технологий и архитектура
* **Язык программирования:** C#
* **UI Фреймворк:** WPF (Windows Presentation Foundation)
* **Платформа:** .NET 10.0-windows
* **База данных:** Microsoft SQL Server
* **Доступ к данным:** ADO.NET (`System.Data.SqlClient`)
* **Архитектура и подходы:** Event-Driven UI, элементы паттерна MVVM, использование сторонних библиотек (MaterialDesignThemes, MahApps.Metro).
